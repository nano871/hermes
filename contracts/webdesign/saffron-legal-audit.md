## PHASE 4: CLAUSE-LEVEL AUDIT — WEB DESIGN AGREEMENT

| Clause | Grade | Issue |
|--------|-------|-------|
| **1.1-1.3 — Scope & Client Responsibilities** | 🟢 SOLID | Clear. Content delivery deadline explicit. |
| **1.4 — Change Orders** | 🟢 SOLID | Email approval for small changes. Written CO for material. |
| **2.1-2.3 — Deposit & Milestones** | 🟢 SOLID | 50% deposit protects against walk. |
| **2.5 — Revisions** | 🟢 SOLID | 2 rounds each. Additional billed. |
| **3.1-3.4 — IP Ownership** | 🟢 SOLID | Three-bucket approach. Template retention clear. |
| **3.5 — AI-Generated Content** | 🟡 THIN | Mentions Thaler but doesn't address: (a) AI image licensing (Midjourney/DALL-E terms changed in 2025-2026), (b) client's obligation to disclose AI use to their users, (c) no-warranty on AI-generated code security |
| **3.6 — Portfolio Rights** | 🟢 SOLID | Three tiers. Good. |
| **3.7 — Credit Link** | 🟢 SOLID | Buyout option. Standard. |
| **4 — Domain & Hosting** | 🟡 THIN | Says designer "may recommend" hosting. Should disclaim liability for hosting provider performance explicitly. Add: Designer is not responsible for hosting security breaches, uptime, or data loss caused by hosting provider. |
| **5.1 — Browser Support** | 🟡 THIN | Good for standard sites. Missing: (a) if client wants IE11/specific legacy browser support, that's additional cost; (b) email client rendering (if building email templates); (c) print CSS scope |
| **5.3(d) — AI Disclaimer** | 🟡 THIN | Covers originality and accuracy. Missing: AI-generated code security disclaimer; FTC endorsement disclosure obligation for AI-generated testimonials/content |
| **6 — Liability Cap** | 🟢 SOLID | Fees paid. Standard. |
| **7 — Indemnification** | 🟡 THIN | Client indemnifies for "content provided by Client." Missing: Client indemnifies for (a) client's use of the site in regulated industries (healthcare, finance, legal) without compliance review; (b) third-party claims from site functionality client requested |
| **9.1 — Termination for Convenience** | 🟡 THIN | Client can terminate with 14 days. Missing: no minimum commitment period. A client could sign, get designs, and cancel before paying milestone 2. Add: deposit non-refundable after design work begins; kill fee for work completed. |
| **Exhibit A — SOW** | 🟢 SOLID | Comprehensive. Browser matrix, content deadlines, AI disclosure. |
